# 匠心永驻——中国古代建筑成就互动叙事可视化系统技术报告

## 1. 项目概述

### 1.1 项目定位
本项目是一个基于现代Web技术的中国古代建筑文化传承与教育平台，通过三维可视化、物理仿真、AI生成等技术手段，将传统建筑智慧以互动体验的形式呈现给用户。

### 1.2 核心价值
- **文化传承**：数字化保护和传播中国古代建筑技艺
- **教育创新**：通过交互式3D模型深度理解建筑原理
- **技术融合**：传统文化与现代技术的完美结合

## 7. AI集成与GLB生成系统

### 7.1 TRELLIS 2D→3D生成管道

**AI生成3D模型管道**
```python
# TRELLIS API调用流程
def generate_glb(client: Client, image_path: str, output_glb_path: str):
    # 1. 图片预处理
    compressed_path = compress_image(image_path, max_size=512)
    image_file = handle_file(compressed_path)
    
    # 2. 启动TRELLIS会话
    session_result = client.predict(api_name="/start_session")
    
    # 3. 2D→3D转换（核心AI推理）
    image_to_3d_result = client.predict(
        api_name="/image_to_3d",
        image=image_file,
        multiimages=[],
        **GENERATION_PARAMS
    )
    
    # 4. 提取GLB模型
    extract_result = client.predict(api_name="/extract_glb", **GLB_PARAMS)
    
    # 5. 保存到项目
    shutil.copy2(str(generated_glb), str(output_glb_path))
```

**生成参数配置**
```python
GENERATION_PARAMS = {
    "seed": 42,                    # 随机种子

    "ss_guidance_strength": 7.5,   # 结构引导强度
```

### 7.2 GLB模型集成系统

**动态模型加载**
```typescript
// GLB模型组件
function PalaceGLBModel() {
  const { scene } = useGLTF(GLB_MODEL_PATH);
  const groupRef = useRef<THREE.Group>(null);
  
  // 自动居中和缩放
  useEffect(() => {
    if (scene && groupRef.current) {
      const box = new THREE.Box3().setFromObject(scene);
      const center = box.getCenter(new THREE.Vector3());
      const size = box.getSize(new THREE.Vector3());
      
      scene.position.copy(center).multiplyScalar(-1);
      const maxDim = Math.max(size.x, size.y, size.z);
      groupRef.current.scale.setScalar(8 / maxDim);
    }
  }, [scene]);
  
  // 缓慢自转展示
  useFrame((state, delta) => {
    if (groupRef.current) {
      groupRef.current.rotation.y += delta * 0.2;
    }
  });
  
  return (
    <Center>
      <group ref={groupRef}>
        <primitive object={scene} />
      </group>
    </Center>
  );
}
```
