# 匠心永驻 - 中国古代建筑成就互动可视化系统

> 2026 全国大学生计算机设计大赛参赛作品  
> 参赛类别：人工智能应用与信息可视化设计（第6类）

## 📖 项目简介

本项目是一个基于 Web 技术的中国古代建筑成就互动可视化系统，通过 3D 建模、力学模拟、数据可视化等技术，展示中国古代建筑的精湛工艺和科学智慧。

### 核心特色

- 🏗️ **精美 3D 建模**：基于真实测绘数据的斗拱、赵州桥、故宫角楼模型
- 🔧 **力学原理展示**：榫卯节点半刚性阻尼器、桥梁应力分布可视化
- 📊 **数据可视化**：ECharts 交互式图表、建筑演化时间轴
- 🎮 **交互式体验**：可点击拆解、着力点选择、实时应力计算
- 📱 **AR 体验**：支持手机扫码 AR 查看古建筑模型
- 🎨 **中式暗黑 UI**：金色装饰、水墨风格、古典美学

## 🚀 快速启动

### 方法一：双击启动脚本（推荐）

1. 找到项目根目录下的 `启动项目.bat` 文件
2. 双击运行
3. 等待自动安装依赖（首次运行）
4. 浏览器将自动打开 `http://localhost:3000`

### 方法二：命令行启动

```bash

# 2. 启动开发服务器
npm run dev

# 3. 浏览器访问
# http://localhost:3000
```

## 📦 技术栈

### 核心框架
- **React 18** - 前端框架
- **Vite** - 构建工具
- **TypeScript** - 类型安全

### 3D 渲染
- **Three.js** - 3D 图形库
- **@react-three/fiber** - React Three.js 渲染器
- **@react-three/drei** - Three.js 辅助工具

### 数据可视化
- **ECharts** - 图表库
- **echarts-for-react** - React ECharts 封装

### 其他技术
- **GSAP** - 动画库
- **@google/model-viewer** - AR 体验
- **Tailwind CSS** - 样式框架
- **React Router** - 路由管理

## 🎯 功能模块

### 1. 首页
- 项目介绍
- 功能导航卡片
- GSAP 动画效果

### 2. 斗拱解析
- **精美 3D 建模**：梯形斗、弧形拱、昂嘴、椽条
- **爆炸拆解**：点击查看内部结构
- **构件高亮**：点击查看详细信息
- **应力热力图**：实时显示受力情况
- **17 个构件**：完整的七铺作双抄双下昂斗拱

### 3. 桥梁抗震（赵州桥）
- **精确建模**：37.02m 跨径，7.23m 矢高
- **28 道分券**：纵向分券纹理
- **4 个敞肩小拱**：精确尺寸
- **交互式着力点**：点击桥面 11 个位置
- **实时应力图**：基于影响线原理计算

### 4. 皇宫游览（故宫角楼）
- **220 攒斗拱**：下层 72、中层 60、上层 28
- **76 条脊**：十字脊屋顶
- **20 根檐柱**：侧脚结构
- **鎏金宝顶**：1.9m 高
- **点击交互**：9 个结构层级详情

### 5. 榫卯力学
- **半刚性阻尼器原理**：3D 动画演示
- **4 种模拟模式**：静态、竖向压力、水平地震、转动耗能
- **能量耗散曲线**：ECharts 实时图表
- **力学可视化**：箭头、转动弧线

### 6. 建筑演化时间轴
- **ECharts 时间轴**：隋-宋-元-明-清
- **5 个关键节点**：赵州桥、应县木塔、营造法式、故宫角楼、颐和园
- **点击交互**：查看详细信息

### 7. AR 体验
- **model-viewer**：支持 AR 模式
- **手机扫码**：移动端 AR 查看
- **4 个模型**：斗拱、赵州桥、故宫角楼、民居

## 📝 使用说明

### 斗拱解析
1. 点击"爆炸拆解"查看内部结构
2. 点击任意构件查看详细信息
3. 调整载荷滑块查看应力变化

### 赵州桥
1. 点击桥面金色圆点选择着力点
2. 调整载荷滑块（0-100 kN）
3. 观察应力图实时变化

### 故宫角楼
1. 点击 3D 模型任意部分
2. 或点击右侧结构图例
3. 查看该层级详细信息

### 榫卯力学
1. 选择模拟模式（静态/竖向/水平/转动）
2. 观察 3D 动画演示
3. 查看能量耗散曲线
